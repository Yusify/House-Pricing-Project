# Boston House Price Prediction

Predicting house prices using Linear Regression.

## Dataset
Boston Housing Dataset — 506 houses, 13 features.

## Steps
- EDA & Visualization
- Correlation Matrix
- VIF — Multicollinearity Check
- Feature Engineering (rm_lstat)
- Linear Regression, Ridge, Lasso
- Streamlit Web App

## Result
RMSE: 4.32 | R²: 0.746
